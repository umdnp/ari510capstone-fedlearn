# Notebooks

This directory contains Jupyter notebooks for the federated learning project.

## Structure

- `exploratory/` - Exploratory Data Analysis (EDA) notebooks for understanding the eICU dataset
- `experiments/` - Experimental notebooks for testing federated learning approaches
- `demos/` - Demo notebooks for presentations and documentation

## Naming Convention

Use descriptive names with dates or version numbers:
- `01_eda_patient_demographics.ipynb`
- `02_feature_engineering.ipynb`
- `exp_01_baseline_model.ipynb`
- `demo_federated_training.ipynb`
